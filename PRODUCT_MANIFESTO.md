# Arcana Product Manifesto

## Vision
Arcana is the **digital hearth** of our home. It connects Sylvia, Aliou, Sacha, and Lisa through shared organization and delightful moments.

## Design Principles

### 1. Apple-like Simplicity (Sylvia Approved)
- Clean, whitespace-heavy interfaces
- No visual clutter
- If it needs a manual, it's broken

### 2. shadcn/ui Native Only
- **Zero custom CSS** - all styling via CSS variables
- Components used as-is from shadcn/ui
- White-label ready for future CMS theming

### 3. Vibe Engineering
- Smooth transitions (framer-motion)
- Skeleton loaders, not spinners
- Confetti on achievements

### 4. Kids Mode (Sacha 9, Lisa 6)
- Toggle for simplified UI
- Larger touch targets
- Fun animations on success
- Age-appropriate language

### 5. Zero-Debt
- TypeScript strict mode
- Zod validation everywhere
- Clean Architecture patterns
