# User Stories

## Epic: Arcana Books (MVP)

### US1: Magic Scan
**As** Aliou/Sylvia
**I want to** take a photo of a book spine or cover
**So that** the system automatically identifies it via Gemini and fills in the details (Author, Publisher, Cover).

### US2: Intelligent Inventory
**As** a user
**I want to** see my books in a clean grid with statuses (Read, To Read, Reading)
**So that** I know what we have at a glance.
*Acceptance Criteria*:
- Clean Grid Layout
- Status badges
- Duplicate detection signal

### US3: Loan Tracking
**As** Sylvia
**I want to** mark a book as "Lent to X"
**So that** we stop losing books to friends.

### US4: AI Reading Cards
**As** a parent
**I want to** get a generated summary/insight card after marking a book as "Read"
**So that** I can discuss it with the kids or recall key points.

### US5: Kids Mode
**As** Sacha/Lisa
**I want to** switch to a fun, simplified interface
**So that** I can explore my library without complex menus.
*Acceptance Criteria*:
- Toggle button
- Larger touch targets
- Success animations (confetti) on finishing a book.
